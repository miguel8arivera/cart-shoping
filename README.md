# Cart Shopping
## Description
project for shopping cart using reactjs and redux and redux-toolkit
## Installation
```bash
yarn add redux react-redux
```
## Usage
```javascript
import { Provider } from 'react-redux'
import store from './store'
ReactDOM.render(
  <Provider store={store}>
    <App />
  </Provider>,
  document.getElementById('root')
)
```
